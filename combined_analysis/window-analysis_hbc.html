This notebook will detail the use of bayesian linear models and mixed models to quantify the difference between single and multi-bat call properties. The final results are presented in the Parameter Summary. This notebook analyses the difference in:

  1. Window received level
  2. FM lower frequency
  3. Dominant frequency range

The linear mixed models used in the following analysis used a non-informative prior. All intervals reported here are 95% highest posterior density compatibility intervals (Gelman and Greenland 2019; McElreath 2020). The compatibility interval corresponds to the estimate-interval that best explains the current dataset.

To read the summarised results straight away, proceed to the Summary Tables.

Received level : RMS

And now build the model that will account for the expected auto-correlation in segments, windows and time in general. This model has \(\sqrt{dBrms}\) as the outcome and group_status as the fixed effect, with recording hour and video annotation ID as random intercepts.

# all dbrms are negative values -- convert them to +ve to allow square rooting
obs_reclevel$sqrtdbrms <- sqrt(abs(min(obs_reclevel$dbrms))+obs_reclevel$dbrms)

obsrec.m1 <- lmer(sqrtdbrms ~ group_status + (1|rechour)+(1|video_annot_id), data=obs_reclevel)

raneffects.m1 <- ranef(obsrec.m1)$video_annot_id$'(Intercept)'
raneffects2.m1 <- ranef(obsrec.m1)$rechour$'(Intercept)'
par(mfrow=c(2,3))
scatter.smooth(obs_reclevel$centred_posix_minute,resid(obsrec.m1))
scatter.smooth(obs_reclevel$group_status,resid(obsrec.m1))
qqnorm(resid(obsrec.m1));qqline(resid(obsrec.m1))
qqnorm(raneffects.m1);qqline(raneffects.m1)
qqnorm(raneffects2.m1);qqline(raneffects2.m1)

obsrec.m1.sim <- sim(obsrec.m1, n.sim=num_post)
m1.hpd.fixefs <- apply(fixef(obsrec.m1.sim),2,get.hpd)

# remember to square the predictions to get the units back to dB from sqrt(dB)
single.reclevel <- fixef(obsrec.m1.sim)[,1]^2-abs(min(obs_reclevel$dbrms))
multi.reclevel <- (fixef(obsrec.m1.sim)[,1]+fixef(obsrec.m1.sim)[,2])^2- abs(min(obs_reclevel$dbrms))
hpd.single.reclevel <- round(get.hpd(single.reclevel),0)
hpd.multi.reclevel <- round(get.hpd(multi.reclevel),0)
# the final expected multi-single rec-level difference in dB
hpd.multi.single.diffrec <- round(get.hpd(multi.reclevel-single.reclevel),1)

The mean received level of single-bat windows is between -33to-32 dB, and multi-bat windows is -33 to -32 dB. The difference betweeen multi-single bat windows is between -0.3 to 0.9 dB.
